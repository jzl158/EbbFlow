# Ebbs & Flows

*Embrace the joys of the day.*

A calming, meditative mobile web game — a self-contained obstacle course of gentle mini-games with singing-bowl tones, glowing aura-ring feedback, and drifting purple/gold/green fireflies.

## Play

Open `index.html` in any browser. It's a single file with no dependencies — works great on a phone.

To serve locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## The journey

1. **Spin to Unlock** — glide your finger around a ring of light
2. **Pick the Flowers** — tap each blossom
3. **Dodge the Lightning** — slide your ship past 10 bolts
4. **Spell Your Name** — tap the floating letters
5. **Avoid the Lava** — trace the winding safe channel
6. **Walk the Plank** — slide up without falling off
7. **Pop the Bubbles** — clear all five

…and a closing message just for you.
